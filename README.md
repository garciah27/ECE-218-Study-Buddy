## What the code does

* Controls the LCD using a keypad
* LCD and keypad are initialized
* Runs using tasks
* There are states that are used to transition between each menu that is displayed on the LCD
* Displays the custom characters 
* User interacts with the keypad by pressing certain inputs that control what gets displayed on the LCD
* LCD shows and keep tracks of the number of hearts and tasks
* LCD also updates number of tasks completed and if any hearts are lost
* Sepearte state for winning therefore there is a different winning animaion
* If player loses, there is a separate losing animation
* User inputs the number of tasks which determines the number of hearts by pressing a number 1-9 on the keypad
* User inputs whether they have finsihed a task by pressing C or D on the keypad
* The user interacts with the menu screens by pressing A or B
* The LCD provides on insructions on how to use the system and interact with the parts
